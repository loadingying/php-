# PHP活动管理系统

## 作者:loadingying

## 使用了bootstrap,zui,mysqli等开源项目

## 使用了阿里云的人机验证

## 数据库版本:5.6.47

## PHP版本:7.2 ,依赖mysqli库

## 初学时写的系统，做得比较差

## 概述：

### 组织:活动以组织为单位进行

### 人员:人员权限有5级，最高两级可以管理任何组织内的事务，接下来两级可以管理组织内事务，最低级只能进行报名签到

### 活动:活动有活动名，发布时间，开始报名时间等等，可以上传附件，签到码随机生成，可进入活动管理页面重新生成

### 活动情况:进入活动管理页面，可查看当前组织人员的报名情况，录取情况，签到情况，并可以更改某些设置
